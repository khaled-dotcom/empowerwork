üìã EVERY TIME YOU WANT TO RUN THE PROJECT:
================================================================================

1. Start MySQL in XAMPP Control Panel (make sure it's green/running)

2. Open PowerShell/CMD and run:

   cd C:\xampp\htdocs\k-main\backend
   .\start.bat

   OR use this single command:
   
   cd C:\xampp\htdocs\k-main
   set PYTHONPATH=.
   python -m uvicorn backend.src.main:app --reload --host 0.0.0.0 --port 8000

3. Backend will start on: http://localhost:8000
   API Docs: http://localhost:8000/docs
